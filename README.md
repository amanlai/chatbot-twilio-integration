# Chatbot SMS
Chatbot SMS Endpoint

## Prerequisite
1.  Python 3.11

## Setup
1.  Clone this repo
2.  Setup python and install deps
    ```shell
    python -m venv venv
    source ./venv/bin/activate
    pip install -r ./requirements.txt
    ```
3.  Copy `.env.example` to `.env` and replace with real values

## Running
```shell
chainlit run ./app.py
```
